# Task Manager | Nest GraphQL

## Run in Dev Mode

1. Clone Project 
```
git clone https://github.com/pxnditxyr/task-manager
```

2. Change name of .env.example and modify
```
mv .env.example .env
```

3. Install dependencies
```
npm i
```

4. Run Docker Image
```
docker compose up -d
```

5. Start Project
```
npm run start:dev
```

6. Visit
```
http://localhost:3000/graphql
```
